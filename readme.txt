Embed videos from Flowplayer online video platform directly into your content.

== Description ==
Embed videos from Flowplayer online video platform directly into your content. This plugin includes both Gutenberg block and support for classic editor.

== Installation ==
= Configure API keys =

1. Navigate to Settings -> Flowplayer Embed
2. Find your Site ID and API key from your Flowplayer app workspace settings page. You can follow the link from the settings page.
3. Copy the Site ID and API key to corresponding fields
4. Save settings

== Frequently Asked Questions ==
= Where do I upload, configure and manage my videos and players? =

Log in to your account in flowplayer.com handle all the video management, this plugin is here to help you with the embeds.

= I can't find the videos in my classic editor, where are they? =

To embed video with classic editor click the "Add media" button and from the left side panel choose Add Flowplayer Embed.
